# 🚗 Sistema de Veículos

Bem-vindo(a) ao **Sistema de Veículos**! Este projeto integra um backend em **Spring Boot** com um frontend em **React**, permitindo gerenciar usuários, veículos e vendas de forma prática e organizada.

## ✨ Funcionalidades

- **Usuários** 👤  
  - Cadastro, edição e exclusão de usuários  
  - Listagem completa de usuários  

- **Veículos** 🚘  
  - Cadastro, edição e exclusão de veículos  
  - Listagem de veículos disponíveis  

- **Vendas** 💰  
  - Registro de venda de veículos  
  - Verificação se um veículo já foi vendido  
  - Listagem de veículos comprados por cada usuário  

## 🛠️ Tecnologias utilizadas

- **Backend**: Spring Boot, JPA/Hibernate, H2 Database  
- **Frontend**: React, Material UI, Axios, React Router  
- **Ferramentas**: Maven, VS Code
